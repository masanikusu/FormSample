# 申請データ登録API

- URL: /app/{app_id}

## 呼び出しタイミング
- アクション：申請ボタン押下時

### 処理概要
- 権限チェック
  - 該当ユーザーが作成ロールを持っていること
- 入力チェック
  - 必須項目チェック
- 変更チェック
- データ登録
  - 入力内容を申請データテーブルに保存
  - 入力内容に基づき申請フローを設定
  - ステータスを承認待ちにする

## リクエスト
#### サンプル
```json
{
  "loginUser_id": 1,
  "app_date": "2024-03-01",
}
```


## レスポンス
```json
{
  "app_id": 1,
  "app_no": "test-No-001"
}
```
